# UI Design Specification: Nexus Boardroom (Polymath Edition)

**Role**: Senior UI/UX Designer & Design Systems Architect.
**Objective**: Define the visual language, component library, and interaction states for a 2D Holographic Tactical Interface.
**Style**: Cyber-Minimalism / Sci-Fi HUD / Glassmorphism.

---

## 1. Design Tokens (视觉原子)

### 1.1 Color System (Semantic)

Do not use hex codes directly; use these semantic tokens mapped to Tailwind classes.

* **Surface Layers** (Backgrounds):
* `surface-void`: `#020617` (Slate-950) -> *Global Background*
* `surface-glass-1`: `rgba(2, 6, 23, 0.6)` -> *Sidebar / Header*
* `surface-glass-2`: `rgba(15, 23, 42, 0.4)` -> *Agent Cards (Idle)*
* `surface-glass-3`: `rgba(15, 23, 42, 0.7)` -> *Agent Cards (Active)*


* **Content Colors**:
* `text-primary`: `#F1F5F9` (Slate-100) -> *Body Text*
* `text-secondary`: `#94A3B8` (Slate-400) -> *Meta Data / Labels*
* `text-accent`: `#22D3EE` (Cyan-400) -> *Highlights / Cursor*


* **Borders & Separators**:
* `border-subtle`: `rgba(255, 255, 255, 0.08)`
* `border-highlight`: `rgba(255, 255, 255, 0.2)`



### 1.2 Agent Theme System (关键)

Each agent ID must map to a specific color profile. Use these for borders, icons, and glow effects.

| Agent ID | Role Name | Color Token | Hex | Tailwind Class | Usage |
| --- | --- | --- | --- | --- | --- |
| **critic** | 批判者 | `color-critic` | `#F43F5E` | `rose-500` | Alert / Danger |
| **historian** | 历史学家 | `color-historian` | `#F59E0B` | `amber-500` | Vintage / Time |
| **pragmatist** | 实干家 | `color-pragmatist` | `#10B981` | `emerald-500` | Success / Stable |
| **expander** | 拓展者 | `color-expander` | `#A855F7` | `purple-500` | Magic / Expansion |
| **verifier** | 验证者 | `color-verifier` | `#3B82F6` | `blue-500` | Security / Trust |
| **synthesizer** | 综合者 | `color-synthesizer` | `#FFFFFF` | `white` | Final / Pure |

### 1.3 Typography (字体系统)

* **Font Family A (Data/Headers)**: `JetBrains Mono` or `Space Mono`.
* Use for: Agent Names, Timestamps, Status, Input Text.
* Transformation: Uppercase for headers.


* **Font Family B (Reading)**: `Inter` or `System UI`.
* Use for: The actual chat content generated by agents (Markdown).



### 1.4 Glassmorphism Recipes (材质配方)

Strictly follow these CSS compositions to achieve the "Holographic" look.

* **Recipe A: "Base Glass" (Sidebar/HUD)**
* `backdrop-filter: blur(12px)`
* `background: linear-gradient(180deg, rgba(255,255,255,0.03) 0%, rgba(255,255,255,0.01) 100%)`
* `border: 1px solid rgba(255,255,255,0.05)`


* **Recipe B: "Holo-Card" (Agent Containers)**
* `backdrop-filter: blur(20px)`
* `background: rgba(15, 23, 42, 0.3)`
* `border: 1px solid rgba(255,255,255,0.08)`
* `box-shadow: 0 4px 30px rgba(0, 0, 0, 0.1)`
* *Top Highlight*: Add an inner top border `border-t: 1px solid rgba(255,255,255,0.2)` to simulate thickness.


* **Recipe C: "Active Glow" (Agent Speaking)**
* Add to Recipe B: `box-shadow: 0 0 40px -10px rgba([AGENT_COLOR], 0.3), inset 0 0 20px rgba([AGENT_COLOR], 0.1)`
* Change Border: `border-color: rgba([AGENT_COLOR], 0.4)`

## 2. Component Specifications (组件规范)

### 2.1 Global Viewport (视口)

* **Z-Layer -1 (Video)**: Fullscreen video. `mix-blend-mode: screen`. `opacity: 0.6`.
* **Z-Layer 0 (Texture)**: Fullscreen `noise.png`. `opacity: 0.05`. `pointer-events: none`.
* **Z-Layer 1 (Vignette)**: Radial gradient overlay: `transparent` center to `rgba(2,6,23,0.8)` edges.

### 2.2 Holo-Card (Agent Card)

The core component.

* **Dimensions**:
* Min-Width: 320px.
* Max-Width: 600px.
* Height: Auto (Fit content) or Fixed 500px with internal scroll.


* **Header**:
* Height: 48px.
* Layout: Flex Row (Space Between).
* Left: Icon (16px) + Agent Name (Mono, Uppercase, Tracking-widest).
* Right: Status Indicator (Pulse Dot).


* **Body**:
* Padding: 24px.
* Font: Inter, 14px, Leading-relaxed (1.6).
* Color: `text-primary` (opacity 0.9).


* **Footer**:
* Height: 32px.
* Border-Top: `1px solid border-subtle`.
* Content: Token Count / Latency (Mono, 10px, `text-secondary`).



### 2.3 Command Console (Input)

* **Shape**: Pill / Rounded Rect (`rounded-xl`).
* **Position**: Fixed bottom (24px from bottom), Centered.
* **Size**: Width 600px (Desktop), Height 64px.
* **Style**: Recipe A (Base Glass) + `bg-gradient-to-r from-cyan-900/20 to-purple-900/20`.
* **Elements**:
* Input Field: Transparent, No border. `text-lg`.
* Visualizer: A canvas layer *behind* the text or *above* the box showing a waveform line.



### 2.4 Sidebar (Navigation)

* **State**: Collapsed (Width 64px) -> Expanded (Width 240px).
* **Style**: Recipe A (Base Glass).
* **Border**: Right border only (`border-white/5`).
* **Items**:
* Active Item: Left-border `2px solid cyan-400`, `bg-white/5`.
* Inactive Item: `opacity 0.6` -> Hover `opacity 1`.

## 3. Dynamic Layout Logic (动态布局逻辑)

Based on the `meta` event from the backend, the grid must adapt.

* **Stage Container**: `display: grid`, `gap: 24px`, `max-width: 1400px`, `margin: auto`.

| Agent Count | Grid Columns (CSS) | Alignment | Notes |
| --- | --- | --- | --- |
| **1** | `grid-cols-1` | Center | Max-width 600px card. |
| **2** | `grid-cols-2` | Center | The classic "Duel" layout. |
| **3** | `grid-cols-3` | Center | Full width usage. |
| **4** | `grid-cols-2` | Center | 2x2 Grid. |
| **5+** | `grid-cols-3` | Start | Masonry or wrapping grid. |

* **Special Rule**: If `agent.id === 'synthesizer'`, it forces a new row and spans full width (`col-span-full`).

## 4. Interaction & Motion (动效规范)

### 4.1 Entry Transitions (Stagger)

When the "Director" sets the stage:

1. **Sidebar**: Slide X (0 -> 100%), Duration 0.5s.
2. **Holo-Cards**: Slide Y (50px -> 0), Opacity (0 -> 1).
* *Stagger*: Each card delays by 0.1s.
* *Spring*: Stiffness 100, Damping 20.



### 4.2 Streaming Effect (Typewriter)

As text arrives via SSE:

* **Cursor**: A blinking block `█` (`animate-pulse`) at the end of the text.
* **Glow**: The card border brightness pulses (opacity 0.3 -> 0.6) on every new chunk received.

### 4.3 Hover Parallax (3D Fake)

Apply strictly to `Holo-Card`.

* **Transform**: `perspective(1000px) rotateX(var(--rX)) rotateY(var(--rY))`.
* **Limit**: Max rotation 5 degrees. Subtle is better.
